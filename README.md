# Comparação — respostas rápidas para o README do projeto_Apps

Este repositório consolida respostas às dúvidas levantadas no README do projeto_Apps, baseando-se nos materiais já incluídos aqui (`readme_v3`, `readmevev2` e `formacao-exit-ready`).

## 1) O que preciso fazer com o sistema?
- Usar o **MAESTRO** como orquestrador: ele recebe o comando (ex.: `/combo [termo]`) e coordena os agentes de validação, research, personas e produção.
- Sempre passar pela **validação** primeiro: só avançar se a ideia tiver demanda, público pagante e urgência.
- Aplicar **Auto-QA** após cada entrega (research, personas, LP, social, ads, email, VSL, webinar) para manter padrão de ~95% de qualidade.
- Iterar rápido com o **EDITOR RÁPIDO** (/refazer-headline, /trocar-estilo, /testar-variacao) sem refazer tudo.

## 2) O que é cada versão?
- **V3 (Maestro V3)**: inclui validação inicial, Auto-QA automatizado, edição granular em minutos e versionamento (v1.0 → v1.1 → v1.2).
- **V2 / Light V2**: versão enxuta focada em landing pages em ~40 min, com Research + Personas + LP + QA mini. Útil para lançamentos rápidos ou MVP.

## 3) Processo do início ao fim
1. **/combo [termo]** → inicia fluxo.
2. **VALIDATOR** confirma mercado, urgência, poder de compra e concorrência; sugere pivôs se reprovar.
3. **Perguntas rápidas** (ticket, depoimentos, diferencial).
4. **RESEARCH** com dados reais, concorrentes, dores, objeções. Auto-QA valida.
5. **PERSONAS** (3 perfis distintos com dores, linguagem, objeções e como vender). Auto-QA valida.
6. **PRODUÇÃO**: LP + Social + Ads + Email + VSL + Webinar para cada persona. Auto-QA em cada entrega.
7. **VERSIONAMENTO**: salva v1.0; iterações criam v1.1, v1.2 ou variações A/B (v1.1a, v1.1b).
8. **ITERAR/EXPORTAR**: ajustes rápidos e exportação (Hotmart/Kiwify/HTML) mantendo histórico.

Use os arquivos de apoio:
- `readme_v3`: visão completa do Maestro V3 e comandos principais.
- `readmevev2`: guia rápido do Maestro Light V2.
- `formacao-exit-ready/`: exemplo completo (research, personas, LP, QA, briefing) para copiar/adaptar.
